What changes

Everything that isn't the content, gone

One keystroke moves a working page into a presenting page. Nothing about your data changes — it is the same tool, with the furniture removed.

1

Full screen, properly

Real browser full screen, with the site header, footer, page title and option bar hidden. On the diagram tool the editor pane collapses too, handing the canvas about a third more width.

2

Readable from the back

Editor type scales up automatically, and the zoom control takes it further. Diagrams are SVG, so they re-fit to the screen and stay sharp at any size.

3

Show only what matters

Focus mode leaves the result alone on screen, edge to edge. The Panels menu toggles any individual pane if you would rather keep the input visible while you talk through it.

The presenter toolbar

Controls where a presenter expects them

A floating toolbar sits at the bottom of the screen and fades out when you stop moving the mouse, so it is there when you reach for it and invisible when you don't.

Open a tool and present →
  • Zoom and fit. Scale the diagram, or the editor type on text tools. Fit re-frames the whole document in one click.
  • Laser pointer. A red pointer with a fading trail, and a ripple where you click — visible in screen shares, because it is drawn into the page.
  • Panels. Show or hide any pane mid-demo without leaving presentation mode.
  • Navigation. Pan the canvas with the arrow buttons or the arrow keys.

Questions

Frequently asked

Which tools support presentation mode?

All fourteen. Every tool page has a Present button in the header, and the behaviour adapts to the tool: the diagram page gives the canvas the whole screen, while the converters scale their type up and can show only the result.

Do I need to install anything?

No. Presentation mode is part of the site and works in any modern browser. Installing JSON Studio as an app is optional, and useful mainly for presenting somewhere with no connection.

Will my diagram stay sharp on a projector?

Yes. Diagrams are SVG, so they are resolution-independent — zooming in on a 4K projector or a 1080p meeting-room screen keeps the edges crisp, and the diagram re-fits automatically when the window changes.

Is the laser pointer visible to people watching a screen share?

Yes. It is drawn into the page itself rather than being an operating-system cursor effect, so it is captured by Zoom, Teams, Meet and any other screen sharing.

Can I hide the input and show only the result?

That is Focus mode — press O, or use the Focus button. Everything except the result disappears, edge to edge. The Panels menu gives finer control if you want to keep one specific pane.
